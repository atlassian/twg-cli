# Contributing

Thanks for helping improve TWG CLI's public docs, examples, recipes, and agent
workflows.

This repository is public. Do not include:

- tokens, credentials, cookies, or `.env` files
- private Jira, Confluence, Bitbucket, or customer URLs
- customer data or proprietary content
- internal Atlassian runbooks or implementation details

Good contributions include:

- clearer docs
- safer examples
- agent prompts and recipes
- troubleshooting notes
- public command examples
- bug reports with redacted output
- skill or plugin workflow requests

Implementation changes for TWG CLI are tracked internally by Atlassian. Public
issues may be mirrored internally when they need product or engineering work.
