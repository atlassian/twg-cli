# Roadmap

This roadmap is intentionally public and high level.

## Now

- Public docs and install flow.
- Public GitHub issue and discussion intake.
- Public agent skills.
- Codex and Claude plugin packaging.
- Release notes tied to stable public CLI releases.

## Next

- Automated public GitHub sync from the private release pipeline.
- Public command reference generated from release builds.
- More agent recipes and examples.
- Marketplace submission automation for Codex and Claude plugin artifacts.

## Later

- Public roadmap labels and project board.
- Compatibility guides for more agent hosts.
- Expanded troubleshooting and diagnostics.
